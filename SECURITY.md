# Security Policy

## Scope

This repository is a public reference implementation and UI proof surface. It does not connect to production systems, tenant data, or customer secrets.

## Reporting

If you find a security issue in the code or deployment workflow, report it privately through GitHub Security Advisories or direct maintainer contact before public disclosure.

## Data posture

- sample data only
- no customer records
- no production credentials
- no PHI / PII workflows implied by this demo
